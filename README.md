# Marion-based Morphostructural Descriptor

This repository provides the Python implementation of an interpretable Marion-based morphostructural descriptor for segmented two-dimensional geological pore images.

The descriptor quantifies pore architecture from binary pore--matrix maps by combining apparent two-dimensional porosity, largest-cluster dominance, pore fragmentation, pore-shape irregularity, lacunarity, skeleton density, box-counting fractal dimension, and departure from a compact 1/10 areal reference.

The method is designed for comparative petrography, digital rock analysis, pore-network screening, and interpretable image-based analysis of segmented porous media.
